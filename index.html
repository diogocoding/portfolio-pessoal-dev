Projetos

App de Treino "Fit & Love" (PWA)

Desenvolvi um Progressive Web App (PWA) completo para acompanhamento de treinos e metas nutricionais. O aplicativo permite que o usuário visualize rotinas de treino (ABC, ABCD), salve o progresso de cargas de cada exercício e acompanhe metas de macronutrientes. O app é instalável, funciona offline e utiliza Firebase Firestore para salvar e sincronizar os dados em tempo real.
Tecnologias: HTML5, CSS3, JavaScript (Vanilla JS), Firebase Firestore, PWA.

Lista de Presentes Interativa (Projeto Pessoal)

Aplicação web completa para gerenciar uma lista de presentes, resolvendo o problema de itens repetidos em um evento familiar. O projeto utiliza JavaScript, Firebase Realtime Database (NoSQL) como back-end para atualização em tempo real, e HTML/CSS para uma interface responsiva.

Residência Tecnológica (Porto Digital & SiDi)

Participei do desenvolvimento do DoT, um sistema de ponto em formato de landing page. O projeto teve foco em simplicidade, usabilidade e acessibilidade, sendo uma oportunidade de aprender e aplicar React em um projeto real com prazo definido.

NutriScan PWA (Projeto Acadêmico)

Desenvolvi um Progressive Web App (PWA) que transforma a câmera do celular em um scanner de código de barras para consulta de informações nutricionais, integrado com a API do Open Food Facts. Implementei funcionalidades offline utilizando Service Workers.

Análise de Dados (Portfólio)

Análise Exploratória de Dados (EDA) completa com dados do Titanic para identificar fatores que influenciaram a sobrevivência. Apliquei técnicas de Data Science como limpeza, tratamento de dados ausentes e visualização com Python, Pandas, Scikit-learn, Matplotlib e Seaborn.

Sistemas em C# (.NET) (Estudos)
